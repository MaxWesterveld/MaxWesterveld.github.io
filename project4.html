Incremental Game

A game about collecting resource to unlock progression

Code parts


This code creates a connection to the SQLite database using Unity's Application.streamingassetsPath.

In the SetData() function a connection is being opened with the database. After that it executes a query to select all rows from the 'currency' table and then assigns it to the corresponding variables in the GameManager.

In the Start() function the SetData() is being called which retrieves the data from the database.
