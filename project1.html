Memory Game

A game about cards

Memory is a game where you have a bunch of cards with pictures on them. Each picture is given 2 cards and the goal of the game is to find the cards with the same pictures

Code parts


In the ShuffleCards() function the cards from the stackOfCards are being shuffled in the while loop and generated at random positions as long as there is not already a card at that position. If the position is empty it removes the top card from the stack using the Pop() function and placing it in the placedCards array.
In the PlaceCardsOnField() function the cards that are being put in the placedCards array will be given a card object and script. Then it calculates the position where the card should be placed and set it as the new position for the card object.
